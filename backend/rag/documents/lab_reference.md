# 检验报告参考范围知识库

## 血常规参考范围

血常规（全血细胞计数，CBC）是最基础的检验项目，用于筛查贫血、感染、血液疾病。

**红细胞系**：
- 红细胞计数（RBC）：男性 4.0-5.5×10¹²/L；女性 3.5-5.0×10¹²/L
- 血红蛋白（Hb/HGB）：男性 120-175g/L；女性 110-150g/L；低于正常下限为贫血
- 血细胞比容（HCT）：男性 0.40-0.50；女性 0.37-0.47
- 平均红细胞体积（MCV）：80-100fl；偏低提示缺铁或地中海贫血，偏高提示叶酸/B12缺乏
- 平均红细胞血红蛋白浓度（MCHC）：320-360g/L

**白细胞系**：
- 白细胞计数（WBC）：成人 4.0-10.0×10⁹/L；轻度升高（10-20）常见于细菌感染或应激，显著升高需排除白血病
- 中性粒细胞（NEU%）：50-70%；感染或应激时升高，严重细菌感染时>80%
- 淋巴细胞（LYM%）：20-40%；病毒感染时相对比例升高
- 单核细胞（MON%）：3-8%
- 嗜酸性粒细胞（EOS%）：0.5-5%；升高提示过敏或寄生虫感染

**血小板系**：
- 血小板计数（PLT）：100-300×10⁹/L；低于100×10⁹/L为血小板减少，可能有出血风险；高于400×10⁹/L为血小板增多

## 血生化参考范围

血生化检测肝肾功能、血糖、血脂及电解质，是慢性病管理的核心指标。

**肝功能**：
- 丙氨酸氨基转移酶（ALT）：男性7-56U/L；女性7-45U/L；升高提示肝细胞损伤（肝炎、脂肪肝、药物性肝损伤）
- 天冬氨酸氨基转移酶（AST）：15-40U/L；ALT/AST比值>2提示酒精性肝病
- 总胆红素（TBIL）：5.1-19.0μmol/L；升高见于溶血、肝炎、胆道梗阻
- 白蛋白（ALB）：35-50g/L；持续偏低提示肝功能下降或营养不良
- 碱性磷酸酶（ALP）：45-125U/L；骨病和胆道疾病时升高

**肾功能**：
- 血清肌酐（Scr）：男性62-115μmol/L；女性53-97μmol/L；升高提示肾功能下降
- 血尿素氮（BUN）：2.9-7.1mmol/L；升高可因肾功能减退或高蛋白饮食
- 估算肾小球滤过率（eGFR）：正常≥90ml/min/1.73m²；<60持续3个月以上为慢性肾病（CKD）
- 血尿酸（UA）：男性208-428μmol/L；女性155-357μmol/L；升高为高尿酸血症，>420（男）或>360（女）为痛风风险阈值

**血糖代谢**：
- 空腹血糖（FPG）：3.9-6.1mmol/L；6.1-6.9为空腹血糖受损（糖尿病前期）；≥7.0为糖尿病
- 糖化血红蛋白（HbA1c）：正常<5.7%；5.7-6.4%为糖尿病前期；≥6.5%为糖尿病；反映近2-3个月平均血糖
- 餐后2小时血糖：<7.8mmol/L正常；7.8-11.0为糖耐量受损；≥11.1为糖尿病

**血脂四项**：
- 总胆固醇（TC）：理想<5.2mmol/L；边缘升高5.2-6.2；高于6.2为高胆固醇血症
- 低密度脂蛋白（LDL-C）：理想<3.4mmol/L；心血管高危人群目标<2.6mmol/L；极高危（既往心梗等）<1.8mmol/L
- 高密度脂蛋白（HDL-C）：男性>1.0mmol/L；女性>1.3mmol/L；越高越好，具有心血管保护作用
- 甘油三酯（TG）：正常<1.7mmol/L；1.7-5.6为轻中度升高；>5.6有急性胰腺炎风险

**电解质**：
- 钾（K⁺）：3.5-5.5mmol/L；低于3.5为低钾（乏力、心律失常），高于5.5为高钾（肾衰患者需注意）
- 钠（Na⁺）：135-145mmol/L；钙（Ca²⁺）：2.1-2.6mmol/L

## 尿常规参考范围

尿常规用于筛查泌尿系感染、肾脏病变和代谢异常。

**外观**：正常尿液淡黄色、清亮；浑浊可能提示感染；红色/棕色尿提示血尿或肌红蛋白尿。

**化学检测**：
- 蛋白质（PRO）：正常阴性（-）；出现（+）及以上为蛋白尿，提示肾小球损伤（糖尿病肾病、肾炎）
- 葡萄糖（GLU）：正常阴性；出现阳性（尿糖）通常提示血糖>9.0mmol/L（肾糖阈），见于糖尿病未控制
- 白细胞（LEU）：正常阴性；阳性提示泌尿系感染，结合亚硝酸盐阳性更有意义
- 亚硝酸盐（NIT）：正常阴性；阳性提示革兰氏阴性菌（大肠杆菌等）感染
- 隐血（BLD）：正常阴性；阳性须结合镜检区分红细胞性血尿与肌红蛋白尿

**镜检（若进行）**：
- 红细胞（RBC）：正常0-3个/高倍视野（HPF）；>3/HPF为镜下血尿；均一型红细胞提示下尿路出血（结石、肿瘤），变形红细胞提示肾小球源性出血
- 白细胞（WBC）：正常0-5/HPF；>5/HPF结合感染症状提示泌尿系感染
- 管型：透明管型少量属正常；出现颗粒管型、红细胞管型提示肾实质损伤

**尿pH**：正常4.5-8.0；持续酸性尿（pH<5）见于痛风、糖尿病；持续碱性尿（>7.5）见于泌尿系感染（产脲酶细菌）。

## 常见异常指标临床意义

**血红蛋白偏低（贫血）**：
- 轻度贫血（Hb 90-110/120g/L）：可出现乏力、头晕、活动后心悸
- 中度贫血（Hb 60-90g/L）：需查明原因并治疗，常见原因为缺铁（铁蛋白<12μg/L）、叶酸/B12缺乏、慢性失血
- 重度贫血（Hb<60g/L）：需住院治疗，可能须输血
- 缺铁性贫血特征：小细胞低色素性贫血（MCV<80fl，MCHC<320g/L），血清铁蛋白降低

**白细胞升高（白细胞增多症）**：
- WBC 10-20×10⁹/L（轻中度）：多为细菌感染、应激反应；结合临床症状判断
- WBC>30×10⁹/L：须排除白血病，建议血液科就诊

**ALT/AST升高（转氨酶升高）**：
- 轻度升高（正常值1-3倍）：可见于脂肪肝、药物反应、剧烈运动；建议戒酒、停可疑药物、复查
- 中度升高（3-10倍）：提示明显肝细胞损伤，须结合病毒性肝炎标志物（乙肝五项、丙肝抗体）进一步检查
- 重度升高（>10倍）：提示急性肝炎或药物性肝损伤，须及时就医

**肌酐升高（肾功能下降）**：
- 轻度升高（115-200μmol/L男性）：eGFR 60-89，CKD 2期；控制血压血糖，避免肾毒性药物，3-6个月复查
- 中度升高（200-440μmol/L）：eGFR 30-59，CKD 3期；肾内科专科随访，低蛋白饮食（每日0.6-0.8g/kg）
- 重度升高（>707μmol/L）：可能需透析治疗，须肾内科住院评估

**血脂异常处理原则**：
- LDL-C升高：调整饮食（减少饱和脂肪和反式脂肪）；若存在心血管风险须启动他汀类药物治疗
- TG>5.6mmol/L：严格低脂饮食、戒酒、考虑贝特类药物；有急性胰腺炎风险，须禁食油腻食物
- HDL-C偏低：规律有氧运动（最有效的提高HDL-C手段）、戒烟、适量红酒（争议，酒精本身有害不建议为此饮酒）
