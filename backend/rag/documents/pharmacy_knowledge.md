# 药品与用药知识库

## 常见OTC非处方药指南

### 解热镇痛类
**布洛芬（Ibuprofen）**
- 适应症：发热（体温≥38.5℃）、头痛、牙痛、肌肉酸痛、关节痛、痛经
- 成人用法：每次200-400mg，每4-6小时一次，每日最多1200mg（OTC剂量）
- 餐后服用减少胃刺激；孕晚期禁用；有消化性溃疡者禁用
- 代表药：芬必得、美林、护彤（儿童）

**对乙酰氨基酚（Paracetamol/Acetaminophen）**
- 适应症：发热、头痛、感冒症状、轻中度疼痛
- 成人用法：每次325-650mg，每4-6小时一次，每日不超过4000mg（严格限制）
- 肝功能不全者慎用；服药期间严禁饮酒（酒精大幅增加肝毒性）
- 警惕：许多复方感冒药已含此成分，避免重复用药超量
- 代表药：泰诺、必理通、百服宁

**阿司匹林（小剂量抗血板）**
- 低剂量（75-100mg/日）：心血管疾病一级/二级预防，需医生指导
- 高剂量：解热镇痛（已逐渐被布洛芬/对乙酰氨基酚替代）
- 禁忌：儿童病毒性感染（Reye综合征风险）、消化性溃疡、出血倾向

## 常见感冒用药

### 感冒症状对症药
**鼻塞/流涕**
- 氯苯那敏（扑尔敏）：第一代抗组胺药，止流涕，有嗜睡副作用，驾车者慎用
- 盐酸伪麻黄碱：鼻腔减充血剂，缓解鼻塞；高血压患者慎用
- 氯雷他定（开瑞坦）：第二代抗组胺药，少嗜睡，适合白天使用

**咳嗽**
- 右美沙芬：中枢性止咳，适用于干咳
- 氨溴索/溴己新：祛痰药，适用于痰多粘稠
- 愈创木酚甘油醚：祛痰药
- 痰液稀薄者使用止咳药，痰多者使用祛痰药，两者区分使用

**复方感冒药注意事项**
- 日夜百服宁、白加黑、泰诺感冒等复方制剂同时含多种成分
- 避免与同类单方药重复使用，尤其是对乙酰氨基酚成分
- 仅缓解症状，不能缩短病程或杀灭病毒

## 常见慢性病用药指导

### 高血压用药
**降压药物分类**
- ACEI类：卡托普利、依那普利（干咳副作用多见）
- ARB类：氯沙坦、缬沙坦（副作用少，替代ACEI）
- CCB类：氨氯地平、硝苯地平（常用，有踝部水肿副作用）
- β受体阻滞剂：美托洛尔、比索洛尔（哮喘患者禁用）
- 利尿剂：氢氯噻嗪（常与其他药物联合）

**用药原则**
- 所有降压药均为处方药，需凭处方购买
- 不得自行停药或减量（可导致血压反弹）
- 每日固定时间服药，通常早晨（CCB类）
- 目标血压：一般人群<140/90mmHg；糖尿病患者<130/80mmHg

### 糖尿病用药
**口服降糖药**
- 二甲双胍：2型糖尿病首选，餐中或餐后服用减少胃肠道反应
- 格列本脲/格列齐特（磺脲类）：有低血糖风险，老年人慎用
- 阿卡波糖：随第一口饭嚼服，控制餐后血糖

**用药注意**
- 均为处方药，需医生指导调整剂量
- 二甲双胍：造影检查前后48小时停药；eGFR<30禁用
- 低血糖急救：立即补充15-20g快速糖（4-5块方糖或150ml果汁）

## 药物相互作用常见警示

### 高风险组合（应避免）
- **布洛芬 + 华法林**：增加出血风险（严重，禁止自行联用）
- **布洛芬 + 阿司匹林**：相互降效，增加胃肠道副作用
- **对乙酰氨基酚 + 酒精**：严重肝毒性风险
- **他汀类 + 大量葡萄柚汁**：升高他汀血药浓度，增加肌病风险
- **降压药 + 布洛芬（长期）**：布洛芬可削弱降压效果

### 特殊人群用药警示
**孕妇**
- 孕早中期可在医生指导下使用对乙酰氨基酚
- 孕晚期（≥28周）禁用NSAIDs（包括布洛芬、阿司匹林）
- 大多数抗生素、中成药需医生评估后才能使用

**儿童**
- 12岁以下儿童禁用阿司匹林（Reye综合征）
- 退热首选对乙酰氨基酚（适用年龄≥3个月）或布洛芬（≥6个月）
- 剂量必须按体重计算，切勿使用成人剂量

**老年人**
- 肾功能下降影响药物排泄，需关注剂量
- 多重用药风险高，用药前告知所有在用药物
- 避免苯二氮䓬类安眠药（跌倒风险）

## 用药安全常识

### 正确用药行为
- 严格按说明书或医嘱用药，不擅自增减剂量
- 用药前检查有效期
- 多数口服药用温开水送服（不用茶水/牛奶/果汁）
- 漏服处理：想起时立即补服；若接近下次用药时间则跳过，不可双倍剂量

### 药品储存
- 大多数药品在室温（10-30℃）、避光干燥处保存
- 胰岛素未开封冰箱（2-8℃）保存；开封后室温可用28-30天
- 眼药水开封后一般4周内用完
- 儿童接触不到的地方存放

### 何时需要就医而非自行用药
- 发烧超过3天，或体温超过39.5℃
- 症状持续超过7天未改善
- 出现皮疹、严重过敏反应（荨麻疹、喉咙肿胀、呼吸困难）→ 立即就医
- 处方药不得自行购买使用
