🏥 大健康智能管家 (Health AI Assistant) V2.0

大健康智能管家是一款深度结合 Agentic UX（智能体交互） 的国民级 AI 健康陪伴平台。它彻底打破了传统医疗 App 繁琐的层级菜单，以自然语言对话为核心底座，无缝串联起“预问诊、挂号、买药、查医保、看报告”等全链路医疗场景。

✨ 核心设计理念

🤖 Agentic UX (智能体驱动)：弱化传统 TabBar，采用“对话框即主界面”。用户的自然语言输入（如“我有点失眠”、“帮我查医保余额”）会由后台智能体自动解析，并路由直达对应功能模块。

👵 适老化底线 (Elderly-First)：全局提供一键切换的「长辈模式」。开启后自动全局放大字号、加粗字体、增强对比度，并优先引导语音交互，跨越银发族的数字鸿沟。

🎨 医疗科技美学 (Medical Tech & Wellness)：采用专业且富有生机的 湖蓝 (Teal) / 翠绿 (Emerald) 作为品牌主色调，传递宁静、治愈与专业的视觉感受。

🚀 核心功能模块

1. 全局交互底座 (Agent UI)

多模态输入：支持文本、语音、图片上传（扫药盒/拍报告）。

动态情境感知：输入框上方会根据当前时间或用户画像，动态生成推荐追问（如“想开点中药调理身体挂什么科？”）。

2. AI 诊室 (智能预问诊)

严格的状态机引导：分为“选咨询人 -> 描述症状 -> 诊疗建议”三步走，约束 AI 幻觉，确保合规。

家庭档案切换：支持为本人或父母等家庭成员建立独立健康档案。

免责与合规声明：在显著位置提示 AI 不替代真实医生面诊。

3. 健康小目标 (数据看板)

穿戴设备联动感知：展示步数、睡眠、心率、活动热量等核心数据。

每日打卡系统：包含喝水、吃药、运动等任务进度追踪，增强用户粘性。

4. 药管家 (医药闭环)

视觉双核引擎：

拍药盒：调用视觉模型进行 OCR，实现用药科普与全网比价。

扫追溯码：扫码识别药品 20 位追溯码，实现防伪验真与过期预警。

用药提醒：以高对比度卡片形式展示近期服药计划，支持一键“已打卡”。

5. 医疗保障管家 (查医保)

国家医保电子凭证：高保真模拟医保码展示逻辑，支持余额“小眼睛”隐私遮挡。

多维服务入口：集成消费记录、年度缴费、共济管理与异地备案快速通道。

扫码支付：一键调起支付条码展示，模拟线下药店/医院结算场景。

6. 就医服务 & 检查报告夹

智能导诊挂号：支持搜索全国三甲医院，结合 LBS 定位推荐附近医疗机构并计算距离。

化验单 AI 解读：支持拍照上传血常规、CT 等报告，利用大模型自动提取异常指标并给出生活建议。

🛠 技术栈说明

当前原型基于纯前端技术栈构建，以单文件组件（Single File Component）的形式实现了所有交互与 UI 展现，便于快速预览与迭代。

框架: React 18+

样式: Tailwind CSS (通过 utility classes 实现极速响应式布局与长辈模式覆写)

图标: Lucide React (轻量且高度一致的开源图标库)

动效: Tailwind animate-in 工具类及 CSS 关键帧动画 (如扫码扫描线)

📦 如何运行与本地开发

环境准备：确保您已安装 Node.js。

初始化项目：推荐使用 Vite 快速创建 React 项目。

npm create vite@latest health-ai-app -- --template react
cd health-ai-app
npm install


安装依赖：安装本项目必需的图标库和 Tailwind CSS。

npm install lucide-react
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p


配置 Tailwind：在 tailwind.config.js 中配置您的路径。

植入代码：将本原型生成的 App.jsx 代码复制并覆盖到项目中的 src/App.jsx。

启动项目：

npm run dev


🔒 合规与安全提示 (上线须知)

本代码仅为前端交互原型，若用于实际商用开发，需严格遵守国家医疗互联网合规要求：

资质要求：在线问诊需依托实体医疗机构获取《互联网医院执业许可》。

算法备案：包含大模型对话功能的产品需进行《深度合成服务算法备案》。

数据安全：医保和患者隐私数据需遵守《个人信息保护法》及医疗数据出境限制，通信过程必须使用强加密。
